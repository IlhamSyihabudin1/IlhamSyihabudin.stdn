Materi Pemrograman Web

Minggu 1

Visual Studio Code (VS Code) adalah editor kode sumber yang ringan dan gratis, dikembangkan oleh Microsoft. Editor ini sangat populer di kalangan pengembang web dan pemrogram karena memiliki berbagai fitur yang membantu dalam pengembangan perangkat lunak.

XAMPP adalah perangkat lunak yang digunakan untuk membuat server lokal di komputer. XAMPP dikembangkan oleh Apache Friends dan berisi berbagai komponen yang diperlukan untuk menjalankan aplikasi berbasis web.

Minggu 2

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dasar sebuah halaman web. HTML bukanlah bahasa pemrograman, tetapi bahasa yang digunakan untuk menyusun elemen-elemen dalam sebuah halaman web, seperti teks, gambar, video, tabel, dan tautan. HTML bekerja bersama dengan CSS (Cascading Style Sheets) untuk tampilan dan desain, serta JavaScript untuk menambahkan interaktivitas.

Minggu 3

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang digunakan untuk mengembangkan aplikasi web dinamis. Dengan PHP, kita bisa membuat website yang dapat berinteraksi dengan pengguna, mengolah data, dan terhubung dengan database seperti MySQL.

Minggu 4

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang digunakan untuk membuat aplikasi web dinamis. PHP dapat memproses data di server sebelum mengirimkan hasilnya dalam bentuk HTML ke browser pengguna. PHP sering digunakan dalam kombinasi dengan HTML, CSS, JavaScript, dan MySQL untuk membangun website interaktif, seperti sistem login, formulir pendaftaran, blog, dan e-commerce.

Minggu 5

Mengontrol alur program dalam PHP sangat penting untuk membuat aplikasi yang lebih interaktif dan dinamis. Kita bisa menggunakan:

Percabangan (if-else, switch-case) untuk pengambilan keputusan.

Perulangan (for, while, do-while, foreach) untuk eksekusi berulang.

Perulangan (Loop) adalah konsep dalam pemrograman yang memungkinkan eksekusi suatu blok kode berulang kali berdasarkan kondisi tertentu. Perulangan berguna untuk menghemat waktu dan mengurangi kode yang berulang.
